# Análise de Design - Boldway Agency

## 1. Design e Layout Geral

### Estrutura da Página

* **Layout**: Grid moderno com seções assimetricas e quebra de containers

* **Navegação**: Header fixo com menu hambúrguer mobile e links desktop

* **Container**: Uso inteligente de containers fluidos com overflow visível

* **Espaçamento**: Generoso uso de whitespace (padding/margin amplos)

### Seções Principais

1. **Hero Section**: Full-screen com vídeo/animação de fundo
2. **About/Agency**: Grid de cards com hover effects
3. **Services**: Seção com ícones minimalistas e descrições
4. **Portfolio**: Galeria masonry com filtros categorias
5. **Team**: Cards circulares com informações de membros
6. **Testimonials**: Carousel com depoimentos em cards
7. **Contact**: Formulário integrado com mapa/informações

## 2. Paleta de Cores e Estilo Visual

### Cores Primárias

* **Preto**: #000000 (fundo principal)

* **Branco**: #FFFFFF (texto principal)

* **Cinza**: #888888, #CCCCCC (elementos secundários)

### Cores de Acabamento

* **Dourado/Amarelo**: #FFD700 (acentos premium)

* **Verde Neon**: #00FF00 (elementos CTAs - adaptável para verde contábil)

* **Gradientes**: Uso sutil de overlays em imagens

### Estilo Visual

* **Minimalista**: Poucos elementos visuais, foco em tipografia

* **Premium**: Uso de espaço e tipografia para transmitir valor

* **Moderno**: Animações suaves e interações hover sofisticadas

## 3. Tipografia e Hierarquia de Textos

### Fontes

* **Títulos**: Sans-serif bold (possivelmente Inter, Poppins ou similar)

* **Corpo**: Sans-serif regular/light

* **Tamanhos**: Grande escala tipográfica (H1: 64px+, H2: 48px+)

### Hierarquia

* **H1**: Títulos de seção com font-weight 700-900

* **H2**: Subtítulos com font-weight 600-700

* **Body**: Texto corrido com font-weight 300-400

* **Small**: Textos auxiliares com font-weight 300

## 4. Elementos de UI/UX

### Botões

* **Estilo**: Outline com hover fill ou solid com gradiente

* **Border-radius**: 0px (quadrado) ou 4px (minimal)

* **Padding**: Generoso (16px+ horizontal, 12px+ vertical)

* **Transições**: Smooth hover com transform ou color change

### Cards

* **Sombra**: Sutil ou nenhuma (flat design)

* **Border**: Mínima ou nenhuma

* **Hover**: Scale transform ou overlay color

* **Padding**: 24px+ interno

### Animações

* **Scroll-trigger**: Fade-in/up ao scrollar

* **Hover**: Scale, color change, underline

* **Loading**: Skeleton screens ou spinners minimalistas

* **Parallax**: Sutil em backgrounds

## 5. Seções e Organização

### Grid System

* **Desktop**: 12-colunas com breaks assimétricos

* **Tablet**: 8-colunas

* **Mobile**: 4-colunas single column

### Ordem Visual

1. Impacto visual imediato (hero)
2. Credibilidade (about/clients)
3. Serviços/produtos
4. Prova social (testimonials)
5. Call-to-action (contact)

## 6. Responsividade e Mobile

### Breakpoints

* **Desktop**: 1200px+

* **Tablet**: 768px-1199px

* **Mobile**: 320px-767px

### Adaptações Mobile

* **Menu**: Hamburguer com slide-in

* **Typography**: Redução de 20-30% nos tamanhos

* **Spacing**: Redução de 40-50% nos paddings

* **Touch**: Botões maiores (44px+ altura mínima)

## 7. Elementos Adaptáveis para Site Contábil

### Mantendo Identidade Verde

* **Verde Principal**: #2E8B57 (Sea Green) ou #228B22 (Forest Green)

* **Verde Claro**: #90EE90 (Light Green) para acentos

* **Verde Escuro**: #006400 para textos importantes

### Elementos Premium Adaptáveis

1. **Hero Section**: Vídeo de escritório ou animação de gráficos contábeis
2. **Service Cards**: Cards para serviços (imposto de renda, abertura de empresa, etc.)
3. **Team Section**: Mostrar contadores e especialistas
4. **Testimonials**: Depoimentos de clientes satisfeitos
5. **Contact**: Formulário com campos específicos (CNPJ, tipo de serviço)

### Adaptações Específicas

* **Ícones**: Usar ícones de calculadora, gráficos, documentos

* **Imagens**: Substituir por imagens de escritórios, reuniões, documentos

* **CTAs**: "Fazer Cotação", "Agendar Consulta", "Abrir Empresa"

* **Trust Badges**: Selos de certificações, parcerias contábeis

### Sugestões de Implementação

* **Color Scheme**: Verde (#228B22) + Branco + Cinza (#F5F5F5)

* **Typography**: Inter ou Roboto para modernidade

* **Layout**: Manter estrutura minimalista mas com elementos de confiança

* **Animations**: Focar em transições suaves e profissionais
